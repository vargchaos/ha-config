🏠 Home Assistant — конфігурація
Особиста конфігурація Home Assistant для розумного будинку.
Включає автоматизації, сценарії, кастомні компоненти та налаштування інтеграцій для керування освітленням, кліматом, безпекою та енергоспоживанням.
Конфігурація підтримується в актуальному стані та синхронізується автоматично через Git Exporter.
